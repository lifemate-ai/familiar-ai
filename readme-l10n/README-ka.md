## рЃарЃљ рЃерЃћрЃБрЃФрЃџрЃўрЃљ

- ­ЪЉЂ **рЃ«рЃћрЃЊрЃЋрЃљ** Рђћ рЃўрЃдрЃћрЃЉрЃА рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃА Wi-Fi PTZ рЃЎрЃљрЃЏрЃћрЃарЃўрЃЊрЃљрЃю рЃљрЃю USB рЃЋрЃћрЃЉрЃАрЃљрЃўрЃЌрЃўрЃЊрЃљрЃю
- ­Ъћё **рЃњрЃљрЃарЃерЃћрЃЏрЃЮ рЃДрЃБрЃарЃћрЃЉрЃљ** Рђћ рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃгрЃарЃћрЃА рЃЊрЃљтђЙрЃЋрЃљ рЃЏрЃбрЃЋрЃарЃўрЃЮрЃА рЃњрЃљрЃарЃћрЃЏрЃЮрЃА
- ­Ъд┐ **рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃљ** Рђћ рЃарЃЮрЃЉрЃЮрЃбрЃўрЃА рЃЋрЃљрЃЎрЃБрЃБрЃЏрЃўрЃЌ рЃбрЃарЃўрЃљрЃџрЃћрЃЉрЃА рЃЮрЃЌрЃљрЃ«рЃерЃў
- ­ЪЌБ **рЃАрЃљрЃБрЃЉрЃљрЃарЃў** Рђћ рЃАрЃљрЃБрЃЉрЃарЃЮрЃЉрЃА ElevenLabs TTS-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ
- ­ЪјЎ **ЛЂл╗ЛЃЛѕл░ЛѓЛї** Рђћ рЃљрЃБрЃЊрЃўрЃЮ рЃўрЃюрЃърЃБрЃбрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃ«рЃЏрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў ElevenLabs Realtime STT (рЃарЃЌрЃБрЃџрЃў)
- ­ЪДа **рЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃЮрЃЉрЃљ** Рђћ рЃљрЃЦрЃбрЃўрЃБрЃарЃљрЃЊ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЊрЃљ рЃўрЃ«рЃАрЃћрЃюрЃћрЃЉрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃћрЃЉрЃА рЃАрЃћрЃЏрЃљрЃюрЃбрЃўрЃЎрЃБрЃарЃў рЃФрЃўрЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ (SQLite + embeddings)
- ­ЪФђ **рЃњрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЌрЃћрЃЮрЃарЃўрЃљ** Рђћ рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃАрЃ«рЃЋрЃўрЃАрЃў рЃърЃћрЃарЃАрЃърЃћрЃЦрЃбрЃўрЃЋрЃљ рЃърЃљрЃАрЃБрЃ«рЃўрЃА рЃЏрЃўрЃфрЃћрЃЏрЃљрЃЏрЃЊрЃћ

## рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА

familiar-ai рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА [ReAct](https://arxiv.org/abs/2210.03629) рЃфрЃўрЃЎрЃџрЃўрЃЌ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃерЃћрЃюрЃА рЃЏрЃўрЃћрЃа рЃљрЃарЃЕрЃћрЃБрЃџрЃў LLM. рЃўрЃА рЃљрЃдрЃўрЃЦрЃЋрЃљрЃЏрЃА рЃЏрЃАрЃЮрЃцрЃџрЃўрЃЮрЃерЃў рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ, рЃцрЃўрЃЦрЃарЃЮрЃЉрЃА рЃарЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃљ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃА Рђћ рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃў.


рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃљрЃарЃљрЃцрЃћрЃарЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА, рЃўрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЏрЃўрЃА рЃерЃўрЃюрЃљрЃњрЃљрЃю рЃАрЃБрЃарЃЋрЃўрЃџрЃћрЃЉрЃќрЃћ: рЃфрЃюрЃЮрЃЉрЃўрЃАрЃЏрЃЮрЃДрЃЋрЃљрЃарЃћрЃЮрЃЉрЃљ, рЃАрЃБрЃарЃЋрЃўрЃџрЃў рЃарЃЮрЃЏ рЃњрЃљрЃарЃћрЃЌ рЃБрЃДрЃБрЃарЃЮрЃА, рЃўрЃАрЃўрЃюрЃўрЃА рЃгрЃљрЃАрЃЋрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћрЃЉрЃљ.

**рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ:**

**рЃљрЃарЃерЃћрЃЏрЃљрЃЊрЃњрЃћрЃюрЃџрЃћрЃЉрЃў:**

| рЃфрЃЋрЃџрЃљрЃЊрЃў | рЃљрЃдрЃгрЃћрЃарЃљ |
|----------|-------------|
| `MODEL` | рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў (рЃњрЃЮрЃюрЃўрЃЋрЃарЃБрЃџрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА) |
| `AGENT_NAME` | TUI-рЃерЃў рЃюрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃў рЃАрЃљрЃ«рЃћрЃџрЃў (рЃЏрЃљрЃњ: `Yukine`) |
| `CAMERA_HOST` | рЃЌрЃЦрЃЋрЃћрЃюрЃў ONVIF/RTSP рЃЎрЃљрЃЏрЃћрЃарЃўрЃА IP рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў |
| `CAMERA_USER` / `CAMERA_PASS` | рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃАрЃћрЃарЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃћрЃЉрЃў |
| `ELEVENLABS_API_KEY` | рЃ«рЃЏрЃўрЃА_OUTPUTрЃА Рђћ [elevenlabs.io](https://elevenlabs.io/) |
| `REALTIME_STT` | `true` рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃў, рЃ«рЃћрЃџрЃБрЃа рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃ«рЃЏрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃерЃћрЃЏрЃЮрЃЊрЃњрЃЮрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ `ELEVENLABS_API_KEY`) |
| `TTS_OUTPUT` | рЃАрЃљрЃЊ рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃЎрЃЋрЃћрЃЌрЃЮрЃА рЃљрЃБрЃЊрЃўрЃЮ: `local` (PC рЃАрЃърЃўрЃЎрЃћрЃарЃў, рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў) \| `remote` (рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃАрЃърЃўрЃЎрЃћрЃарЃў) \| `both` |

## MCP рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃў

familiar-ai рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа [MCP (Model Context Protocol)](https://modelcontextprotocol.io) рЃАрЃћрЃарЃЋрЃћрЃарЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ. рЃћрЃА рЃњрЃљрЃгрЃЋрЃЊрЃћрЃЉрЃљ рЃерЃћрЃюрЃА рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃЮ рЃћрЃЦрЃАрЃбрЃћрЃарЃюрЃБрЃџрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ, рЃцрЃљрЃўрЃџрЃўрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃљ, рЃЋрЃћрЃЉрЃАрЃћрЃюрЃбрЃарЃљрЃџрЃБрЃарЃў рЃФрЃўрЃћрЃЉрЃљ, рЃЌрЃБ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃАрЃ«рЃЋрЃљ рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃА рЃќрЃЮрЃџрЃў.

рЃЊрЃљрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЌ рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃў `~/.familiar-ai.json` (рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ рЃарЃЮрЃњрЃЮрЃарЃф Claude Code):

```json
{
  "mcpServers": {
    "filesystem": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/home/user"]
    },
    "memory": {
      "type": "sse",
      "url": "http://localhost:3000/sse"
    }
  }
}
```

рЃЏ_supрЃБрЃџрЃў рЃбрЃарЃљрЃюрЃАрЃърЃЮрЃарЃбрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў рЃЎрЃЮрЃюрЃЎil╔ЎrрЃЮрЃЉрЃўрЃџрЃўрЃљ:
- **`stdio`**: рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃАрЃБрЃЉрЃърЃарЃЮрЃфрЃћрЃАрЃў (`command` + `args`)
- **`sse`**: HTTP+SSE рЃАрЃћрЃарЃЋрЃћрЃарЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ (`url`)

рЃгрЃљрЃерЃџрЃўрЃџрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљ рЃљрЃарЃўрЃА `MCP_CONFIG=/path/to/config.json`.

## рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃљ

familiar-ai рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌ Рђћ рЃљрЃю рЃљрЃарЃфрЃћрЃарЃЌрЃў рЃЏрЃўрЃќрЃюрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.

| рЃюрЃљрЃгрЃўрЃџрЃў | рЃарЃљ рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА | рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў | рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ? |

> **рЃЎрЃљрЃЏрЃћрЃарЃљрЃљ рЃФрЃџрЃўрЃћрЃарЃў рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃљ**. рЃЏрЃўрЃАрЃў рЃњрЃљрЃарЃћрЃерЃћ, familiar-ai рЃЏрЃљрЃўрЃюрЃф рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃАрЃљрЃБрЃЉрЃљрЃарЃў Рђћ рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЋрЃћрЃа рЃ«рЃћрЃЊрЃљрЃЋрЃА рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃА, рЃарЃљрЃф рЃАрЃБрЃџ рЃЕрЃћрЃЏрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ.

рЃљрЃарЃљрЃцрЃћрЃарЃў рЃњрЃљрЃЦрЃЋрЃА? рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃљрЃљ:

```env
API_KEY=sk-...
```

рЃњрЃљрЃЏрЃљрЃарЃЌрЃћрЃЌ `./run.sh` рЃЊрЃљ рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ рЃАрЃљрЃБрЃЉрЃљрЃарЃў. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃљ рЃърЃарЃЮрЃфрЃћрЃАрЃерЃў.

### Wi-Fi PTZ рЃЎрЃљрЃЏрЃћрЃарЃљ (Tapo C220)

1. Tapo рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃерЃў: **рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў Рєњ рЃдрЃарЃЏрЃљ Рєњ рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў** Рђћ рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў (рЃљрЃа TP-Link рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў)
2. рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃћрЃЌ рЃЎрЃљрЃЏрЃћрЃарЃўрЃА IP рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃарЃЮрЃБрЃбрЃћрЃарЃўрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃерЃў
3. рЃЊрЃљрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЌ `.env`-рЃерЃў:
   ```env
   CAMERA_HOST=192.168.1.xxx
   CAMERA_USER=your-local-user
   CAMERA_PASS=your-local-pass
   ```

### рЃ«рЃЏрЃљ (ElevenLabs)

1. рЃЏрЃўрЃўрЃдрЃћрЃЌ API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў [elevenlabs.io](https://elevenlabs.io/)
2. рЃЊрЃљрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЌ `.env`-рЃерЃў:
   ```env
   ELEVENLABS_API_KEY=sk_...
   ELEVENLABS_VOICE_ID=...   # рЃљрЃарЃЕрЃћрЃЋрЃўрЃЌрЃў, рЃЌрЃБ рЃљрЃа рЃўрЃЦрЃюрЃљ рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃћрЃцрЃЮрЃџрЃбрЃў рЃ«рЃЏрЃљ
   ```

рЃАрЃърЃћрЃЦрЃбрЃарЃў рЃЮрЃарЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃА `TTS_OUTPUT`:

```env
TTS_OUTPUT=local    # PC рЃАрЃърЃўрЃЎрЃћрЃарЃў (рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў)
TTS_OUTPUT=remote   # ┘Ђ┘ѓпи ┘ЃпД┘Ё┘іп▒пД
TTS_OUTPUT=both     # рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃАрЃърЃўрЃЎрЃћрЃарЃў + PC рЃАрЃърЃўрЃЎрЃћрЃарЃў рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ
```

#### A) рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃАрЃърЃўрЃЎрЃћрЃарЃў (go2rtc-рЃЊрЃљрЃю)

рЃЊрЃљрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЌ `TTS_OUTPUT=remote` (рЃљрЃю `both`). рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃА [go2rtc](https://github.com/AlexxIT/go2rtc/releases):

1. рЃњрЃљрЃЊрЃЏрЃЮрЃљрЃЉрЃарЃФрЃљрЃюрЃћрЃЌ рЃЉрЃўрЃюрЃљрЃарЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ [рЃњрЃљрЃЏрЃЮрЃерЃЋрЃћрЃЉрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃЊрЃљрЃю](https://github.com/AlexxIT/go2rtc/releases):
   - Linux/macOS: `go2rtc_linux_amd64` / `go2rtc_darwin_amd64`
   - **Windows: `go2rtc_win64.exe`**

2. рЃњрЃљрЃюрЃљрЃЌрЃљрЃЋрЃАрЃћрЃЌ рЃЊрЃљ рЃЊрЃљрЃљрЃарЃЦрЃЋрЃљрЃЌ:
   ```
   # Linux / macOS
   ~/.cache/embodied-claude/go2rtc/go2rtc          # chmod +x рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ

   # Windows
   %USERPROFILE%\.cache\embodied-claude\go2rtc\go2rtc.exe
   ```

3. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `go2rtc.yaml` рЃўрЃЏрЃљрЃЋрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў:
   ```yaml
   streams:
     tapo_cam:
       - rtsp://YOUR_CAM_USER:YOUR_CAM_PASS@YOUR_CAM_IP/stream1
   ```
   рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃљрЃюрЃњрЃљрЃарЃўрЃерЃўрЃА рЃАрЃћрЃарЃЌрЃўрЃцрЃўрЃЎрЃљрЃбрЃћрЃЉрЃў (рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ TP-Link рЃдрЃарЃБрЃЉрЃљрЃюрЃўрЃА рЃљрЃюрЃњрЃљрЃарЃўрЃерЃўрЃА рЃЎрЃарЃћрЃЊрЃћрЃюрЃбрЃљрЃџрЃћрЃЉрЃў).

4. familiar-ai рЃўрЃгрЃДрЃћрЃЉрЃА go2rtc-рЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљрЃА рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю. рЃЌрЃБ рЃерЃћрЃюрЃў рЃЎрЃљрЃЏрЃћрЃарЃљ рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљрЃА рЃБрЃгрЃДрЃЋрЃћрЃб рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃ«рЃЏрЃљ (рЃБрЃЎрЃБрЃЏрЃўрЃќрЃћрЃќрЃў), рЃ«рЃЏрЃљ рЃЕрЃљрЃЏрЃЮрЃЊрЃўрЃА рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃАрЃърЃўрЃЎрЃћрЃарЃўрЃЊрЃљрЃю.

#### B) рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў PC рЃАрЃърЃўрЃЎрЃћрЃарЃў

рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў (`TTS_OUTPUT=local`). рЃЏрЃфрЃюрЃћрЃЉрЃБрЃџрЃћрЃЉрЃў рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃљрЃЊ рЃљрЃцрЃўрЃЦрЃАрЃўрЃарЃћрЃЉрЃА: **paplay** Рєњ **mpv** Рєњ **ffplay**. рЃљрЃАрЃћрЃЋрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃБрЃџрЃўfallback, ЯцюЯцг `TTS_OUTPUT=remote` лИ go2rtc лЮлх tersedia.

| OS | рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ |
|----|---------|
| macOS | `brew install mpv` |
| Ubuntu / Debian | `sudo apt install mpv` (лИл╗лИ `paplay` ЛЄлхЛђлхли `pulseaudio-utils`) |
| WSL2 / WSLg | `sudo apt install pulseaudio-utils` Рђћ рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `PULSE_SERVER=unix:/mnt/wslg/PulseServer`-рЃЏрЃљ `.env`-рЃерЃў |
| Windows | [mpv.io/installation](https://mpv.io/installation/) Рђћ рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃЊрЃљ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ PATH-рЃерЃў, **рЃљрЃа** `winget install ffmpeg` |

> рЃЌрЃБ рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃЏрЃћрЃюрЃўрЃА рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћ, рЃАрЃљрЃБрЃЉрЃљрЃарЃў рЃЎрЃЋрЃџрЃљрЃЋ рЃњрЃљрЃ«рЃџрЃљрЃЋрЃЌ Рђћ рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃљрЃа рЃўрЃЌрЃљрЃЏрЃљрЃерЃћрЃЉрЃА.

### рЃ«рЃЏрЃЮрЃЋрЃљрЃюрЃў рЃўрЃюрЃърЃБрЃбрЃў (Realtime STT)

рЃЊрЃљрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЉрЃўрЃА `REALTIME_STT=true`-рЃерЃў `.env` рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃ«рЃћрЃџрЃћрЃЉрЃўрЃА рЃ«рЃћрЃџрЃўрЃЌ рЃ«рЃЏрЃљ рЃўрЃюрЃърЃБрЃбрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА:

```env
REALTIME_STT=true
ELEVENLABS_API_KEY=sk_...   # рЃўрЃњрЃўрЃЋрЃћ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў рЃарЃЮрЃњрЃЮрЃарЃф TTS рЃљрЃЦрЃЋрЃА
```

familiar-ai рЃБрЃАрЃЏрЃћрЃюрЃА рЃЏрЃўрЃЎрЃарЃЮрЃцрЃЮрЃюрЃўрЃА рЃљрЃБрЃЊрЃўрЃЮрЃА ElevenLabs Scribe v2-рЃўрЃА рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃА, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃАрЃљрЃБрЃЉрЃљрЃарЃА рЃгрЃДрЃЋрЃћрЃб. рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃљрЃа рЃљрЃарЃўрЃА. рЃЌрЃљрЃюрЃљрЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃњрЃљрЃЦрЃюрЃћрЃЋрЃўрЃЌрЃў рЃарЃћрЃЪрЃўрЃЏрЃЌрЃљрЃю (Ctrl+T).

## TUI

familiar-ai рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃбрЃћрЃЦрЃАрЃбрЃБрЃа UI, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃдрЃћрЃЉрЃБрЃџрЃЮрЃЉрЃА [Textual](https://textual.textualize.io/):

- рЃЕрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃљрЃЊрЃў рЃўрЃАрЃбрЃЮрЃарЃўрЃљ рЃњрЃљрЃЏрЃЮрЃ«рЃљрЃбрЃЋрЃўрЃА verlopen-рЃгрЃћрЃарЃўрЃџрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў
- Tab-рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ `/quit`, `/clear`
- рЃЊрЃљрЃцрЃўрЃЦрЃАрЃўрЃарЃћрЃЉрЃљ рЃљрЃњрЃћрЃюрЃбрЃўрЃА рЃЏрЃўрЃбрЃЮрЃА рЃЏрЃљрЃЌ рЃЏрЃўрЃЏрЃЮрЃцрЃћрЃюрЃўрЃА рЃЊрЃљрЃАрЃЋрЃћрЃюрЃћрЃЉрЃљрЃЏрЃЊрЃћ
- **рЃАрЃљрЃљрЃЌрЃўрЃА рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў** рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃџрЃљрЃЊ рЃўрЃгрЃћрЃарЃћрЃЉрЃљ `~/.cache/familiar-ai/chat.log`

рЃЮрЃарЃЮ logfile-рЃўрЃА рЃњрЃљрЃЊрЃљрЃ«рЃћрЃЊрЃЮрЃА рЃћрЃарЃЌрЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃљрЃЊ):
```bash
tail -f ~/.cache/familiar-ai/chat.log
```

## рЃ«рЃерЃўрЃарЃљрЃЊ рЃЊрЃљрЃАрЃЏрЃБрЃџрЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃў

**Q: рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ GPU-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ?**  
рЃЊрЃўрЃљрЃ«. рЃЊрЃљрЃЏрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃћрЃџрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃў (multilingual-e5-small) рЃЎрЃљрЃарЃњрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА CPU-рЃќрЃћ. GPU рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ.

**Q: рЃерЃћрЃФрЃџрЃћрЃЉ рЃЌрЃБ рЃљрЃарЃљ рЃАрЃ«рЃЋрЃљ рЃЎрЃљрЃЏрЃљрЃарЃљрЃА рЃарЃЮрЃЏ рЃЊрЃљрЃљрЃЏрЃбрЃЎрЃўрЃфрЃЮ?**  
рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃА рЃЎрЃљрЃЏрЃћрЃарЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЎрЃарЃФрЃљрЃџрЃљрЃЋрЃА ONVIF + RTSP, рЃБрЃюрЃЊрЃљ рЃўрЃЏрЃБрЃерЃљрЃЮрЃА. Tapo C220 рЃљрЃарЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃарЃЕрЃћрЃБрЃџ рЃЏрЃЮрЃЊрЃћрЃџрЃў.

**Q: рЃЕрЃћрЃЏрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃўрЃарЃўрЃфрЃ«рЃБрЃЏрЃќрЃћ?**  
рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў рЃЊрЃљ рЃбрЃћрЃЦрЃАрЃбрЃў рЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ рЃерЃћрЃю рЃЏрЃўрЃћрЃа рЃљрЃарЃЕрЃћрЃБрЃџ LLM API-рЃќрЃћ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋ `~/.familiar_ai/`.

**Q: рЃарЃљрЃбрЃЮрЃЏ рЃгрЃћрЃарЃА рЃљрЃњрЃћрЃюрЃбрЃў `№╝ѕ...№╝Ѕ` рЃАрЃўрЃюрЃБрЃюрЃБрЃарЃљрЃЊ?**  
рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ `ELEVENLABS_API_KEY` рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃњрЃљрЃЦрЃЋрЃЌ. рЃЌрЃБ рЃљрЃарЃљ, рЃ«рЃЏрЃљ рЃњрЃљрЃЌрЃўрЃерЃБрЃџрЃў рЃљрЃЦрЃЋрЃА рЃЊрЃљ рЃљрЃњрЃћрЃюрЃбрЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃњрЃљрЃЊрЃљрЃўрЃЋрЃџрЃўрЃА.

## рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃцрЃЮрЃюрЃЊрЃў

рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ, рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА? рЃўрЃ«рЃўрЃџрЃћрЃЌ [docs/technical.md](./docs/technical.md) familiar-ai рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃЎрЃЋрЃџрЃћрЃЋрЃљрЃА рЃЊрЃљ рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃћрЃЉрЃА Рђћ ReAct, SayCan, Reflexion, Voyager, рЃАрЃБрЃарЃЋрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЊрЃљ рЃЏрЃћрЃбрЃў.

## рЃгрЃЋрЃџрЃўрЃџрЃў

familiar-ai рЃљрЃарЃўрЃА рЃдрЃўрЃљ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў. рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃћрЃА resonates рЃЌрЃЦрЃЋрЃћрЃюрЃАрЃЌрЃЋрЃўрЃА Рђћ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃљрЃЊ рЃЌрЃБ рЃцрЃўрЃџрЃЮрЃАрЃЮрЃцрЃўрЃБрЃарЃљрЃЊ Рђћ рЃгрЃЋрЃџрЃўрЃџрЃў рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ.

**рЃЎрЃљрЃарЃњ рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў рЃЊрЃљрЃАрЃљрЃгрЃДрЃћрЃЉрЃљрЃЊ:**

| рЃбрЃћрЃарЃўрЃбрЃЮрЃарЃўрЃљ | рЃарЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃљрЃљ |
|------|---------------|
| рЃљрЃ«рЃљрЃџрЃў рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃљ | рЃЏрЃћрЃбрЃў рЃЎрЃљрЃЏрЃћрЃарЃћрЃЉрЃўрЃА (RTSP, IP Webcam), рЃЏрЃўрЃЎрЃарЃЮрЃцрЃЮрЃюрЃћрЃЉрЃўрЃА, рЃљрЃЦрЃбрЃБрЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ |
| рЃљрЃ«рЃљрЃџрЃў рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃў | рЃЋрЃћрЃЉрЃАрЃћрЃюрЃбрЃарЃљрЃџрЃБрЃа рЃФрЃўрЃћрЃЉрЃљрЃА, рЃАрЃљрЃ«рЃџрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃљ, рЃЎрЃљрЃџрЃћрЃюрЃЊрЃљрЃарЃў, рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў MCP-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ |
| рЃљрЃ«рЃљрЃџрЃў рЃЉрЃћрЃЦрЃћрЃюрЃЊрЃў | рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў LLM рЃљрЃю рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃў, рЃарЃЮрЃЏ рЃерЃћрЃћрЃцрЃћрЃарЃћрЃЉрЃљ `stream_turn` рЃўрЃюрЃбрЃћрЃарЃцрЃўрЃБрЃА |
| рЃърЃћрЃарЃАрЃЮрЃюрЃљ рЃАрЃћрЃЦрЃарЃћрЃЉрЃћрЃЉрЃў | ME.md рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃърЃўрЃарЃЮрЃЋрЃюрЃћрЃЉрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА |
| рЃЎрЃЋрЃџрЃћрЃЋрЃљ | рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃАрЃБрЃарЃЋрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў, рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃљрЃдрЃЊрЃњрЃћрЃюрЃљ, рЃњрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЌрЃћрЃЮрЃарЃўрЃўрЃА рЃърЃарЃЮрЃЏрЃърЃбрЃўрЃарЃћрЃЉрЃљ |
| рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ | рЃњрЃљрЃгрЃЋрЃЊрЃўрЃџрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ, рЃЏрЃљрЃАрЃбрЃћрЃарЃћрЃЉрЃў, рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃћрЃЉрЃў |

рЃўрЃ«рЃўрЃџрЃћрЃЌ [CONTRIBUTING.md](./CONTRIBUTING.md) рЃњрЃљрЃюрЃЦрЃЏрЃћрЃю рЃАрЃљрЃЊрЃћрЃЋрЃў рЃЊрЃљрЃ»рЃњрЃБрЃцрЃћрЃЉрЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃўрЃЌ, рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃбрЃўрЃџрЃќрЃћ, рЃЊрЃљ PR рЃњрЃћрЃњрЃЏрЃљ.

рЃЌрЃБ рЃљрЃа рЃ«рЃљрЃарЃЌ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃЊ рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃЌ, [рЃдрЃўрЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљрЃќрЃћ](https://github.com/lifemate-ai/familiar-ai/issues) Рђћ рЃЉрЃћрЃЊрЃюрЃўрЃћрЃарЃћрЃЉрЃў рЃЋрЃљрЃарЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃњрЃўрЃЏрЃљрЃарЃбрЃўрЃЋрЃЮрЃЌ.
