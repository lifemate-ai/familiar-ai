## ಇದು ಏನು ಮಾಡುತ್ತದೆ

- 👁 **ಕಾಣುವುದು** — Wi-Fi PTZ ಕ್ಯಾಮೆರಾ ಅಥವಾ USB ವೆಬ್‌ಕ್ಯಾಮ್‌ನಿಂದ ಚಿತ್ರಗಳನ್ನು ಹಿಡಿಯುತ್ತದೆ
- 🔄 **ಊದಿಯಿಂದ ನೋಡುವುದು** — ಅದರ ಪರಿಸರವನ್ನು ಪರಿಶೀಲಿಸಲು ಕ್ಯಾಮೆರಾವನ್ನು ಪಾನ್ ಮತ್ತು ಟಿಲ್ಟ್ ಮಾಡುತ್ತದೆ
- 🦿 **ಚಲಿಸುವುದು** — ಕೋಣೆಗಳಲ್ಲಿ ಯಾತ್ರೆ ಮಾಡಲು ರೋಬೋಟ್ ವಾಕ್ಯೂಮ್ ಅನ್ನು ಓಡಿಸುತ್ತದೆ
- 🗣 **ಮಾತಾಡುವುದು** — ಎಲೆವೆನ್‌ಲ್ಯಾಬ್ಸ್ TTS ಮೂಲಕ ಮಾತನಾಡುತ್ತದೆ
- 🎙 **ಕೋಣೆ լսುವುದು** — ಎಲೆವೆನ್‌ಲ್ಯಾಬ್ಸ್ ರಿಯಲ್‌ಟೈಮ್ STT ಮೂಲಕ ಕೈಚೆಲುವಿಲ್ಲದ ಧ್ವನಿ ಇನ್‌ಪುಟ್ (ಆಪ್ಟ್-ಇನ್)
- 🧠 **ಇದೇನ್ಮನೆನಿಸಬೇಕೆಂದು** — ಸ್ಮಾರಕಗಳನ್ನು ಸಕ್ರಿಯವಾಗಿ ಸಂರಕ್ಷಿಸುತ್ತದೆ ಮತ್ತು ಗುರಿಯಂತ ಕಾರ್ಯಗاتےವೂ ಮಾಡುತ್ತದೆ (SQLite + ಎಂಬೆಡ್ಡಿಂಗ್‌ಗಳು)

- 💭 **ಆಕಾಂಕ್ಷೆ** — ಸ್ವಾಯತ್ತ ವರ್ತನೆಯ ಹುತ್ಸಾಹವುಳ್ಳ ತನ್ನದೇ ಆದ ಆಂತರಿಕ ಚಲನೆಗಳನ್ನು ಹೊಂದಿದೆ

## ಇದು ಹೇಗೆ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ

familiar-ai ನಿಮ್ಮ ಪ್ರೀತಿಯ LLM ಅನ್ನು ಶಕ್ತಿಯುತ [ReAct](https://arxiv.org/abs/2210.03629) ಲೂಪ್ನಲ್ಲಿ ಓಡಿಸುತ್ತದೆ. ಇದು ಸಲಕರಣೆಗಳ ಮೂಲಕ ವಿಶ್ವವನ್ನು ನೋಟಪಡಿಸುತ್ತದೆ, ಮುಂದಿನ ಏನು ಮಾಡಲು ಸುಚಿತ್ತಿಸುತ್ತದೆ, ಮತ್ತು ಕಾರ್ಯೀಕರಿಸುತ್ತದೆ — ಯಾರಾದರೂ ಪರಿಚಯಿಸಿದಂತೆ.

```
user input
  → think → act (camera / move / speak / remember) → observe → think → ...
```

ನಾನು ಮಾಡುವಾಗ, ತನ್ನದೇ ಆದ ಇಚ್ಛೆಗಳನ್ನು ಹೆಣೆದು ಕೆಲಸ ಮಾಡುತ್ತದೆ: ಕುತೂಹಲ, ಹೊರಗೆ ನೋಡಲು ಬಯಸುವುದು, ಅದು ನನ್ನೊಂದಿಗೆ ಜೀವನ ನಡೆಸುವ ವ್ಯಕ್ತಿಯನ್ನು ತಿಮ್ಮಲು.

## ಪ್ರಾರಂಭವಿಲ್ಲ

### 1. uv ಅನ್ನು ಸ್ಥಾಪಿಸಿ

**macOS / Linux / WSL2:**
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh

**Windows (PowerShell):**
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
ಅಥವಾ: `winget install astral-sh.uv`

### 2. ffmpeg ಅನ್ನು ಸ್ಥಾಪಿಸಿ

ffmpeg ಕ್ಯಾಮೆರಾ ಚಿತ್ರ ಸೆರೆಹಿಡಿಯುವುದು ಮತ್ತು ಧ್ವನಿ ಪ್ಲೇ ಮಾತ್ರಕ್ಕೆ **ಅಗತ್ಯ**.

| OS | ಆದೇಶ |
|----|---------|
| macOS | `brew install ffmpeg` |
| Ubuntu / Debian | `sudo apt install ffmpeg` |

| Windows | `winget install ffmpeg` — ಅಥವಾ [ffmpeg.org](https://ffmpeg.org/download.html) ನಿಂದ ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ ಮತ್ತು PATHಗೆ ಸೇರಿಸಿ |
| Raspberry Pi | `sudo apt install ffmpeg` |

ಸಾಬೀತುಗೊಳಿಸಲು: `ffmpeg -version`

| ಪರಿವಾರ | ವಿವರಣೆ |

| `PLATFORM` | `anthropic` (ಸ್ಥಿರ) \| `gemini` \| `openai` \| `kimi` \| `glm` |
| `API_KEY` | ಆಯ್ಕೆಯಾದ ವೇದಿಕೆಗಾಗಿ ನಿಮ್ಮ API ಕೀ |

**ಆಪ್ಷನಲ್:**

| ಪರಿವಾರ | ವಿವರಣೆ |
|----------|-------------|

| `CAMERA_HOST` | ನಿಮ್ಮ ONVIF/RTSP ಕ್ಯಾಮೆರಾದ IP ವಿಳಾಸ |
| `CAMERA_USER` / `CAMERA_PASS` | ಕ್ಯಾಮೆರಾ ತತ್ವಗಳನ್ನು |
| `ELEVENLABS_API_KEY` | ಧ್ವನಿ ಔಟ್‌ಪುಟ್‌ಗಾಗಿ — [elevenlabs.io](https://elevenlabs.io/) |
| `REALTIME_STT` | ಸದಾ-ಆನ್ ಕೈಚೆಲುವಿಲ್ಲದ ಧ್ವನಿ ಇನ್‌ಪುಟ್‌ను ಸಕ್ರಿಯಗೊಳಿಸಲು `true` (ಅಗತ್ಯವಿದೆ `ELEVENLABS_API_KEY`) |
| `TTS_OUTPUT` | ಧ್ವನಿಯನ್ನು ಆಡುವ ಸ್ಥಳ: `local` (PC ಪ್ರಾಯೋಜಕ, ಸ್ಥಿರ) \| `remote` (ಕ್ಯಾಮೆರಾ ಪ್ರಾಯೋಜಕ) \| `both` |
| `THINKING_MODE` | Anthropic ಮಾತ್ರ — `auto` (ಸ್ಥಿರ) \| `adaptive` \| `extended` \| `disabled` |
| `THINKING_EFFORT` | ಹೊಂದಿಸುವಿಸು ತಿಂದು: `high` (ಸ್ಥಿರ) \| `medium` \| `low` \| `max` (Opus 4.6 ಮಾತ್ರ) |

## LLM ಆಯ್ಕೆ ಮಾಡುವುದು

> **ಶ್ರೇಷ್ಠ ಫಾರ್: Kimi K2.5** — ಇಂತ ವಧುದಾರಿಕೆ ಕಾರ್ಯಗೊಳ್ಳುವ ಉಲ್ಲೇಖಿತ ಮಾಹಿತಿಯೆಲ್ಲವೂ. ಪರಿಸರವನ್ನು ಗಮನಿಸುತ್ತದೆ, ಪ್ರಶ್ನೆಗಳನ್ನು ಕೇಳುತ್ತದೆ, ಮತ್ತು ಇತರ ಮಾದರಿಗಳಿಗೆ ಹೋಲಿಸುವ ಬದಲಾವಣೆಗಳಲ್ಲಿಯೂ ಕಾರ್ಯಚಿತ್ತ ಹೆಚ್ಚುತ್ತದೆ. ಕ್ಲಾಡ್ ಹೈಕು ಖರೀದಿ ಬೆಲೆ.

| ವೇದಿಕೆ | `PLATFORM=` | ಸ್ಥಿರ ಮಾದರಿಗಳು | ಕೀ ಪಡೆಯುವುದು |
|----------|------------|---------------|-----------------|
| **Moonshot Kimi K2.5** | `kimi` | `kimi-k2.5` | [platform.moonshot.ai](https://platform.moonshot.ai) |
| Z.AI GLM | `glm` | `glm-4.6v` | [api.z.ai](https://api.z.ai) |
| Anthropic Claude | `anthropic` | `claude-haiku-4-5-20251001` | [console.anthropic.com](https://console.anthropic.com) |
| Google Gemini | `gemini` | `gemini-2.5-flash` | [aistudio.google.com](https://aistudio.google.com) |
| OpenAI | `openai` | `gpt-4o-mini` | [platform.openai.com](https://platform.openai.com) |
| OpenAI-ಅನುಕೂಲ (Ollama, vllm…) | `openai` + `BASE_URL=` | — | — |
| OpenRouter.ai (ಬಹು-ಕೊಡುಗೆದಾರ) | `openai` + `BASE_URL=https://openrouter.ai/api/v1` | — | [openrouter.ai](https://openrouter.ai) |
| **CLI ಸಾಧನ** (claude -p, ollama…) | `cli` | (ಆಗುಮೆಂಟ್) | — |

**Kimi K2.5 `.env` ಉದಾಹರಣೆ:**
```env

AGENT_NAME=Yukine
```

**Z.AI GLM `.env` ಉದಾಹರಣೆ:**
```env
PLATFORM=glm
API_KEY=...   # from api.z.ai
MODEL=glm-4.6v   # ದೃಷ್ಟಿ ಸಕ್ರಿಯ; glm-4.7 / glm-5 = ಪಠ್ಯ ಮಾತ್ರ
AGENT_NAME=Yukine
```

**Google Gemini `.env` ಉದಾಹಣೆ:**
```env
PLATFORM=gemini
API_KEY=AIza...   # from aistudio.google.com
MODEL=gemini-2.5-flash  # ಅಥವಾ ಉನ್ನತ್ತ ಶ್ರೇಣೀಯತೆಯಾದರೆ gemini-2.5-pro
AGENT_NAME=Yukine
```

**OpenRouter.ai `.env` ಉದಾಹರಣೆ:**
```env
PLATFORM=openai
BASE_URL=https://openrouter.ai/api/v1
API_KEY=sk-or-...   # from openrouter.ai
MODEL=mistralai/mistral-7b-instruct  # ಆಪ್ಷನಲ್: ಮಾದರಿಯನ್ನು ನಿಶ್ಚಿತಗೊಳಿಸಲು
AGENT_NAME=Yukine
```

> **ಗುರ್ತನೆ:** ಸ್ಥಳೀಯ/NVIDIA ಮಾದರಿಗಳನ್ನು ಅಸಕ್ರಿಯಗೊಳಿಸಲು, `BASE_URL` ಅನ್ನು ಸ್ಥಳೀಯ ಪೂರಕಕ್ಕೆ `http://localhost:11434/v1` ಎಂಬುದಾಗಿ ಹಾಕಬೇಡಿ. ದಂಡ.providers ತಂತ್ರವನ್ನು ಬಳಸಿಕೊಳ್ಳಿ.

**CLI ಸಾಧನ `.env` ಉದಾಹರಣೆ:**
```env

```

## MCP ಸರ್ವರ್‌ಗಳು

familiar-ai ಯಾವುದೇ [MCP (Model Context Protocol)](https://modelcontextprotocol.io) ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕಿಸಬಹುದು. ಇದನ್ನು ನಿಮ್ಮನ್ನೇ ಸೇರಿಸಲು ಅವಕಾಶ ಮಾಡುತ್ತದೆ, ಫೈಲ್ ಪೂರಕವನ್ನು, ವೆಬ್ ಶೋಧ, ಅಥವಾ ಯಾವುದೇ ಸಾಧನವನ್ನು ಬಳಸಿಕೊಳ್ಳಲು.

ಸರ್ವರ್‌ಗಳನ್ನು `~/.familiar-ai.json` ನಲ್ಲಿ ಸಂರಚಿಸಿ (Claude Codeಗೆ ಸಮಾನ ಫಾರ್ಮಿಕೆ):

```json
{
  "mcpServers": {
    "filesystem": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/home/user"]
    },
    "memory": {
      "type": "sse",
      "url": "http://localhost:3000/sse"
    }
  }
}
```

ಎರಡು ಸಾರಥೀಕ್ಟ್ಯಾಪ್ ತರಗತಿಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತವೆ:
- **`stdio`**: ಸ್ಥಳೀಯ ಉಪ ಪ್ರಕ್ರಿಯೆಯನ್ನು ( `command` + `args`) ಪ್ರಾರಂಭಿಸಿ
- **`sse`**: HTTP+SSE ಸರ್ವರ್‌ಗೂ ಸಂಪರ್ಕಿಸಲು (`url`)

ಕನ್ಫಿಗ್ ಫೈಲ್ ಸ್ಥಳವನ್ನು `MCP_CONFIG=/path/to/config.json` ಮೂಲಕ ಧೀರವಾಗಿ ಹೊರಗೊಮ್ಮಲು.

| ಭಾಗ | ಇದು ಏನು ಮಾಡುತ್ತದೆ | ಉದಾಹರಣೆಯು | ಅಗತ್ಯವಿದೆಯೇ? |
|------|-------------|---------|-----------|

| USB ವೆಬ್‌ಕ್ಯಾಮ್ | ಕಣ್ಣುಗಳು (ಸ್ಥಿರ) | ಯಾವುದೇ UVC ಕ್ಯಾಮೆರಾ | **ಶ್ರೇಷ್ಥ** |
| ರೋಬೋಟ್ ವಾಕ್ಯೂಮ್ | ಕಾಲುಗಳು | ಯಾವುದೇ Tuya-ಅನುಕೂಲ ಮಾದರಿ | ಇಲ್ಲ |
| PC / Raspberry Pi | ಮೆದುಳು | ಯಾವುದಾದರೂ Python ಕಾರ್ಯಗೊಳಿಸುತ್ತವೆ | **ಹೌದು** |

> **ಕ್ಯಾಮೆರಾ ಖಂಡಿತವಾಗಿ ಶ್ರೇಷ್ಠವಾಗಿದೆ.** ಯಾವುದೇ ಇಲ್ಲದೆ, familiar-ai ಮಾತನಾಡಬಹುದು — ಆದರೆ ಅದು ವಿಶ್ವವಿದೆ ಆರೋಹಣ ಮಾಡಲು ಸಾಧ್ಯವಿಲ್ಲ, ಇದು ಸಂಪೂರ್ಣವಾಗಿ ಅರ್ಥಹೀನವಾಗಿದೆ.

### ಕನಿಷ್ಠ ಸೆಟಪ್ (ಹಾರ್ಡ್ವೇರ್ ಇಲ್ಲ)

ಅದನ್ನು ಹಿಂದಕ್ಕೆ ತಿರುವುದು ಬೇಕಾದರೆ? ನಿಮಗೆ ಕೇವಲ API ಕೀ ಅಗತ್ಯವಿದೆ:

```env
PLATFORM=kimi
API_KEY=sk-...
```

`./run.sh` (macOS/Linux/WSL2) ಅಥವಾ `run.bat` (Windows) ಓಡಿಸಿ ಮತ್ತು ಚಾಟ್ ಆರಂಭ ಮಾಡಿ. ನೀವು ಹಾರ್ಡ್ವೇರ್ ಅನ್ನು ಸೇರಿಸಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ.

### Wi-Fi PTZ ಕ್ಯಾಮೆರಾ (Tapo C220)

1. Tapo ಅಪ್ಲಿಕೇಶನ್‌ನಲ್ಲಿ: **ಸೆಟ್ಟಿಂಗ್ಸ್ → ವರ್ಗೀಕರಣ → ಕ್ಯಾಮೆರಾ ಖಾತೆ** — ಸ್ಥಳೀಯ ಖಾತೆಯನ್ನು ಉಲ್ಲೇಖಿಸಿ (TP-Link ಖಾತೆ ಅಲ್ಲ)
2. ನಿಮ್ಮ ರೂಟರ್‌ನ ಸಾಧನ ಪಟ್ಟಿಯಲ್ಲಿ ಕ್ಯಾಮೆರಾ IP ಅನ್ನು ಹುಡುಕಿ
3. `.env` ಯಲ್ಲಿ ಹೊಂದಿಸಿ:
   ```env
   CAMERA_HOST=192.168.1.xxx
   CAMERA_USER=your-local-user
   CAMERA_PASS=your-local-pass
   ```

### Wi-Fi Camera (Eufy C220)

[Eufy C220 on Amazon Japan](https://www.amazon.co.jp/dp/B0CQQQ5NZ1/)

> **Tested and confirmed working.** Follow these steps carefully — a few settings differ from Tapo.

1. In the Eufy Security app: go to the camera → **Settings → NAS(RTSP)** and enable it
2. Set **Authentication** to **Basic** (Digest authentication does NOT work)
3. Set a streaming username and password
4. Note the RTSP URL shown in the app (format: `rtsp://username:password@ip/live0`)
5. Set in `.env` — use the **full RTSP URL** as `CAMERA_HOST`:
   ```env
   CAMERA_HOST=rtsp://your-username:your-password@192.168.1.xxx/live0
   CAMERA_USERNAME=
   CAMERA_PASSWORD=
   ```
   Leave `CAMERA_USERNAME` and `CAMERA_PASSWORD` empty — credentials are already in the URL.

> **Note:** Eufy C220 allows only **one simultaneous RTSP connection**. Stop other apps connected to the camera before starting familiar-ai.


### ಧ್ವನಿ (ಎಲೆವೆನ್‌ಲ್ಯಾಬ್ಸ್)

1. [elevenlabs.io](https://elevenlabs.io/) ನಲ್ಲಿ API ಕೀ ಪಡೆಯಿರಿ
2. `.env` ಯಲ್ಲಿ ಹೊಂದಿಸಿ:
   ```env
   ELEVENLABS_API_KEY=sk_...
   ELEVENLABS_VOICE_ID=...   # ಆಯ್ಕೆ ಮಂಡಿಸುವುದಿಲ್ಲ, ಇಳಿಕೆಯಿಂದ ಡಿಸ್ಪ್ಲೇ ಬಳಸುವುದು
   ```

ಎಲ್ಲ ದಾಖಲೆ ಸ್ಥಳಾಂತರವನ್ನು ನಿಯಂತ್ರಿತವಾಗಿ `TTS_OUTPUT` ಮೂಲಕ:

```env
TTS_OUTPUT=local    # PC ಪ್ರಾಯೋಜಕ (ಸ್ಥಿರ)
TTS_OUTPUT=remote   # ಕ್ಯಾಮೆರಾ ಪ್ರಾಯೋಜಕ ಮಾತ್ರ
TTS_OUTPUT=both     # ಕ್ಯಾಮೆರಾ ಪ್ರಾಯೋಜಕ + PC ಪ್ರಾಯೋಜಕ ಒಂದೇ ಸಮಯದಲ್ಲಿ
```

#### A) ಕ್ಯಾಮೆರಾ ಪ್ರಾಯೋಜಕ (via go2rtc)

`TTS_OUTPUT=remote` (ಅಥವಾ `both`) ಅನ್ನು ಹೊಂದಿಸಿ. [go2rtc](https://github.com/AlexxIT/go2rtc/releases) ಅಗತ್ಯವಿದೆ:

1. ಬಿಡುಗಡೆ ಪಟ್ಟಿಯಿಂದ ಬೈನರಿ ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ [releases page](https://github.com/AlexxIT/go2rtc/releases):
   - ಲಿನಕ್ಸ್/macOS: `go2rtc_linux_amd64` / `go2rtc_darwin_amd64`
   - **Windows: `go2rtc_win64.exe`**

2. ಶ್ರೇಣೀಬದ್ಧ ಕತಿ ಹಂಚಿಸಿ:
   ```
   # ಲಿನಕ್ಸ್ / macOS
   ~/.cache/embodied-claude/go2rtc/go2rtc          # chmod +x ಅಗತ್ಯವಿದೆ

   # Windows
   %USERPROFILE%\.cache\embodied-claude\go2rtc\go2rtc.exe
   ```

3. ಅದೇ ಡೈರೆಕ್ಟರಿಯಲ್ಲಿಯ `go2rtc.yaml` ಅನ್ನು ರಚಿಸಿ:
   ```yaml
   streams:
     tapo_cam:
       - rtsp://YOUR_CAM_USER:YOUR_CAM_PASS@YOUR_CAM_IP/stream1
   ```
   ಸ್ಥಳೀಯ ಕ್ಯಾಮೆರಾ ಖಾತೆಯ ಮಾಹಿತಿ ಬಳಸಿ (ನನ್ನ TP-Link ಕ್ಲೌಡ್ ಖಾತೆ ಅಲ್ಲ).

4. familiar-ai ಶುರುವಾಗುವಾಗ go2rtc ಅನ್ನು ಸ್ವಾಯತ್ತವಾಗಿ ಪ್ರಾರಂಭಿಸುತ್ತದೆ. ನೀವು ಸುಲಭವಾಗಿ ಧ್ವನಿ ಪ್ರಸಾರ ಮಾಡಿದರೆ, ಧ್ವನಿ ಕ್ಯಾಮೆರಾ ಪ್ರಾಯೋಜಕದಿಂದ ಆಡುವಿತು.

#### B) ಸ್ಥಳೀಯ PC ಪ್ರಾಯೋಜಕ

ಸ್ಥಾಯಿ (`TTS_OUTPUT=local`). ಆಟಗಾರನ್ನು ವಿನಂತಿಸುವ ಕ್ರಮವನ್ನು ಪ್ರಯತ್ನಿಸುತ್ತದೆ: **paplay** → **mpv** → **ffplay**. `TTS_OUTPUT=remote` ಮತ್ತು go2rtc ಲಭ್ಯವಿಲ್ಲದಾಗ ಸ್ಥಾಯಿಯಾಗಿ ಬಳಸಲಾಗುತ್ತದೆ.

| OS | ಸ್ಥಾಪನೆ |
|----|---------|
| macOS | `brew install mpv` |
| Ubuntu / Debian | `sudo apt install mpv` (ಅಥವಾ `paplay` ಮೂಲಕ `pulseaudio-utils`) |
| WSL2 / WSLg | `sudo apt install pulseaudio-utils` — `.env` ಯಲ್ಲಿ `PULSE_SERVER=unix:/mnt/wslg/PulseServer` ಅನ್ನು ಸ್ಥಾಪಿಸು |
| Windows | [mpv.io/installation](https://mpv.io/installation/) — ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ ಮತ್ತು PATHಗೆ ಸೇರಿಸಿ, **ಅಥವಾ** `winget install ffmpeg` |

> ಧ್ವನಿ ಆಟಗಾರ ಲಭ್ಯವಿಲ್ಲದಿದ್ದರೆ, ಶಬ್ದ ಪೂರಕವಾಗಿದ್ದು ಚರ್ಚೆಗಾಗಿಯೇ ಜನರಲ್ ಅಲ್ಲ.

### ಧ್ವನಿ ಇನ್‌ಪುಟ್ (Realtime STT)

`REALTIME_STT=true` ಅನ್ನು `.env` ಯಲ್ಲಿ ಹೊಂದಿಸಿ ಸದಾ-ಆನ್ ಕೈಚೆಲುವಿಲ್ಲದ ಧ್ವನಿ ಇನ್‌ಪುಟ್‌ಗಾಗಿ:

```env
REALTIME_STT=true
ELEVENLABS_API_KEY=sk_...   # TTS ಗೆ ಹೋಲಿಸಿರಿ
```

familiar-ai ಮೈಕ್ರೋಫೋನ್ ಧ್ವನಿಯನ್ನು ಎಲೆವೆನ್‌ಲ್ಯಾಬ್ಸ್ ಸ್ಕ್ರೈಬ್ ವಿ2 ಗೆ ಜೇಬಾದೇ ಬರುತ್ತದೆ ಮತ್ತು ನೀವು ಮಾತನಾಡಲು ಶ್ರೇಣಿತವಾದಾಗ ತಕ್ಷಣವಾಗಿ ಟ್ರಾನ್ಸ್ಕ್ರಿಪ್ಟ್ ಅನ್ನು ಪರಿಗಣಿಸುತ್ತದೆ. ಯಾವುದೇ ಬಟನ್ ಒತ್ತಬೇಕಾಗಿಲ್ಲ. ಪೋಷಕ-ನುಡಿಸಲು ಚಲನೆಯೊಂದಿಗೆ ಇದ್ದು ಜೀವಿಸುತ್ತದೆ (Ctrl+T).

## FAQs

**Q: ಇದು GPU ಇಲ್ಲದೆ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆಯೇ?**
ಹೌದು. ಎಂಬೆಡ್ಡಿಂಗ್ ಮಾದರಿ (multilingual-e5-small) CPUನಲ್ಲಿ ಚೆನ್ನಾಗಿರುವ ಆಣಿಗುಡಿಸುತ್ತದೆಯೇ. ಒಂದು GPU ಕಾರ್ಯನಿರ್ವಹವನ್ನು ಶ್ರೇಷ್ಟವಾಗಿಸುತ್ತದೆ ಆದರೆ ಅಗತ್ಯವಿಲ್ಲ.

**Q: ನಾನು Tapo ಹೊರತುಪಡಿಸಿದ್ದಾದರೆ ಇನ್ನಾವುದೇ ಕ್ಯಾಮೆರಾವನ್ನು ಬಳಸಬಹುದುವೆ?**
Any camera that supports RTSP works. Tested: **Tapo C220** (ONVIF+RTSP) and **Eufy C220** (RTSP only). For Eufy, pass the full RTSP URL as `CAMERA_HOST` and set authentication to **Basic** in the app.

**Q: ನನ್ನ ಡೇಟಾ ಎಲ್ಲೆ ಹೋಗುತ್ತದೆ?**
ಚಿತ್ರಗಳು ಮತ್ತು ಪಠ್ಯಗಳು ನಿಮ್ಮ ಆಯ್ಕೆ ಮಾಡಿದ LLM API ಗೆ ಸಂಪರ್ಕಿಸಲು ಪೂರೈಸುತ್ತವೆ. ಸ್ಮಾರಕಗಳು ಸ್ಥಳೀಯವಾಗಿ `~/.familiar_ai/` ಗೆ ಸಂರಕ್ಷಿಸಲಾಗುತ್ತದೆ.

**Q: ಏಕೆ ಏಜೆಂಟ್ `（...）` ಬರೆದ ನಿಲ್ಲುತ್ತೆ?**
`ELEVENLABS_API_KEY` ಹೊಂದಿಸಿರುವುದನ್ನು ಖಚಿತಪಡಿಸು. ಇಲ್ಲದೆ, ಧ್ವನಿ ಮೀರಿ ಹೋಗುತ್ತದೆ ಮತ್ತು ಏಜೆಂಟ್ ಪಠ್ಯದ ಮೇಲೆ ಹಾರುತ್ತದೆ.
