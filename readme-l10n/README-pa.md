## ਇਹ ਕਿਸ ਤਰ੍ਹਾਂ ਕੰਮ ਕਰਦਾ ਹੈ

familiar-ai ਇੱਕ [ReAct](https://arxiv.org/abs/2210.03629) ਲੂਪ ਚਲਾਉਂਦਾ ਹੈ ਜੋ ਤੁਹਾਡੇ ਚੋਣ ਦੇ LLM ਤੋਂ ਸ਼ਕਤੀਸ਼ਾਲੀ ਹੁੰਦਾ ਹੈ। ਇਸਨੂੰ उपकरणਾਂ ਦੁਆਰਾ ਦੁਨੀਆ ਦੀ ਸਮਝ ਮਿਲਦੀ ਹੈ, ਅਗਲੇ ਕੀ ਕਰਨ ਬਾਰੇ ਸੋਚਦੀ ਹੈ, ਅਤੇ ਕੰਮ ਕਰਦੀ ਹੈ — ਜਿਵੇਂ ਕਿ ਇਕ ਵਿਅਕਤੀ ਕਰੇਗੀ।

```
user input
  → think → act (camera / move / speak / remember) → observe → think → ...
```

ਜਦੋਂ Idle ਹੁੰਦਾ ਹੈ, ਇਹ ਆਪਣੇ ਆਪ ਦੀਆਂ ਇਛਾਵਾਂ 'ਤੇ ਕੰਮ ਕਰਦਾ ਹੈ: ਜਿਗਿਆਸਾ, ਬਾਹਰ ਦੇਖਣ ਦੀ ਇਛਾ, ਆਪਣੇ ਜੀਵਨ ਸਾਥੀ ਦੀ ਯਾਦ ਕਰਨਾ।

### 1. uv ਇੰਸਟਾਲ ਕਰੋ

**macOS / Linux / WSL2:**
```bash

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

## MCP ਸਰਵਰ

familiar-ai ਕਿਸੇ ਵੀ [MCP (ਮਾਡਲ ਸੰਦਰਭ ਪ੍ਰੋਟੋਕੋਲ)](https://modelcontextprotocol.io) ਸਰਵਰ ਨਾਲ ਜੁੜ ਸਕਦੀ ਹੈ। ਇਹ ਤੁਹਾਨੂੰ ਬਾਹਰੀ ਯਾਦ, ਫਾਈਲ ਸਿਸਟਮ ਪ੍ਰਵേശ ਅਤੇ ਵੈਬ ਖੋਜ ਜਾਂ ਕਿਸੇ ਹੋਰ ਟੂਲ ਵਿੱਚ ਪਲੱਗ ਕਰਨ ਦੀ ਆਗਿਆ ਦਿੰਦੀ ਹੈ।

ਸਰਵਰਾਂ ਨੂੰ `~/.familiar-ai.json` ਵਿੱਚ ਸੰਰਚਨਾ ਕਰੋ (Claude ਕੋਡ ਦੇ ਸਮਾਨ ਫਾਰਮੈਟ):

```json
{
  "mcpServers": {
    "filesystem": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/home/user"]
    },
    "memory": {
      "type": "sse",
      "url": "http://localhost:3000/sse"
    }
  }
}
```

ਦੋ ਆਵਿਰਕ ਕਿਸਮਾਂ ਦਾ ਸਮਰਥਨ ਕੀਤਾ ਜਾਂਦਾ ਹੈ:
- **`stdio`**: ਇਕ ਸਥਾਨਕ ਸਬ-ਪ੍ਰਕਿਰਿਆ ( `command` + `args`) ਸਿੱਧਾ ਲਾਂਚ ਕਰੋ
- **`sse`**: ਇਕ HTTP+SSE ਸਰਵਰ ਨਾਲ ਜੁੜੋ (`url`)

ਕੰਫਿਗਰੇਸ਼ਨ ਫਾਈਲ ਦਾ ਸਥਾਨ `MCP_CONFIG=/path/to/config.json` ਨਾਲ ਓਵਰਰਾਈਡ ਕਰੋ।

| ਹਿੱਸਾ | ਇਹ ਕੀ ਕਰਦਾ ਹੈ | ਉਦਾਹਰਣ | ਲੋੜੀਂਦੇ? |
|------|-------------|---------|-----------|
| Wi-Fi PTZ ਕੈਮਰਾ | ਅੱਖਾਂ + ਗਰਦਨ | Tapo C220 (~$30, Eufy C220) | **ਸਿਫਾਰਿਸ਼ ਕੀਤੀ** |
| USB ਵੈਬਕੈਮ | ਅੱਖਾਂ (ਫਿਕਸਡ) | ਕੋਈ ਵੀ UVC ਕੈਮਰਾ | **ਸਿਫਾਰਿਸ਼ ਕੀਤੀ** |

> **ਇੱਕ ਕੈਮਰਾ ਬਹੁਤ ਮਲਕੀਅਤ ਦੀ ਸਿਫਾਰਿਸ਼ ਕੀਤੀ ਜਾਂਦੀ ਹੈ।** ਬਿਨਾਂ ਕਿਸੇ ਕੈਮਰੇ ਦੇ, familiar-ai ਫਿਰ ਵੀ ਗੱਲ ਕਰ ਸਕਦੀ ਹੈ — ਪਰ ਇਸ ਨੂੰ ਦੁਨਿਆ ਨਹੀਂ ਦੇਖ ਸਕਦੀ, ਜੋ ਕਿ ਸਾਰਾ ਮੁੱਦਾ ਹੈ।

ਕੇਵਲ ਕੌਸ਼ਲ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹੋ? ਤੁਹਾਨੂੰ ਕਿਵੇਂ API ਦੀ ਕੀਮਤ ਦੀ ਲੋੜ ਹੈ:

`./run.sh` (macOS/Linux/WSL2) ਜਾਂ `run.bat` (Windows) ਚਲਾਓ ਅਤੇ ਗੱਲਬਾਤ ਸ਼ੁਰੂ ਕਰੋ। ਜੋੜਨ ਲਈ ਹਾਰਡਵੇਅਰ ਇਸਬਾਦ ਦੇ ਜਿਥੇ।

### Wi-Fi PTZ ਕੈਮਰਾ (Tapo C220)

1. Tapo ਐਪ ਵਿੱਚ: **ਸੈਟਿੰਗਜ਼ → ਉੱਚਿਤ → ਕੈਮਰਾ ਖਾਤਾ** — ਇੱਕ ਸਥਾਨਕ ਖਾਤਾ ਬਣਾਓ (TP-Link ਖਾਤਾ ਨਹੀਂ)
2. ਆਪਣੇ ਰਾਊਟਰ ਦੇ ਦੇਵਾਈਸ ਸੂਚੀ ਵਿੱਚ ਕੈਮਰੇ ਦੀ IP ਲੱਸੀ ਗਈ
3. `.env` ਵਿੱਚ ਸੈਟ ਕਰੋ:
   ```env
   CAMERA_HOST=192.168.1.xxx
   CAMERA_USER=your-local-user
   CAMERA_PASS=your-local-pass
   ```

### Wi-Fi Camera (Eufy C220)

[Eufy C220 on Amazon Japan](https://www.amazon.co.jp/dp/B0CQQQ5NZ1/)

> **Tested and confirmed working.** Follow these steps carefully — a few settings differ from Tapo.

1. In the Eufy Security app: go to the camera → **Settings → NAS(RTSP)** and enable it
2. Set **Authentication** to **Basic** (Digest authentication does NOT work)
3. Set a streaming username and password
4. Note the RTSP URL shown in the app (format: `rtsp://username:password@ip/live0`)
5. Set in `.env` — use the **full RTSP URL** as `CAMERA_HOST`:
   ```env
   CAMERA_HOST=rtsp://your-username:your-password@192.168.1.xxx/live0
   CAMERA_USERNAME=
   CAMERA_PASSWORD=
   ```
   Leave `CAMERA_USERNAME` and `CAMERA_PASSWORD` empty — credentials are already in the URL.

> **Note:** Eufy C220 allows only **one simultaneous RTSP connection**. Stop other apps connected to the camera before starting familiar-ai.


### ਆਵਾਜ਼ (ElevenLabs)

1. [elevenlabs.io](https://elevenlabs.io/) ਤੇ ਇੱਕ API ਦੀ ਕੀਮਤ ਪ੍ਰਾਪਤ ਕਰੋ
2. `.env` ਵਿੱਚ ਸੈਟ ਕਰੋ:
   ```env
   ELEVENLABS_API_KEY=sk_...
   ELEVENLABS_VOICE_ID=...   # ਵਿਕਲਪਕ, ਜੋ ਛੱਡਣ 'ਤੇ ਡਿਫਾਲਟ ਆਵਾਜ਼ ਵਰਤੋਂ ਕਰਦਾ ਹੈ
   ```

ਦੋ ਪਲੇਅਬੈਕ ਗੰਤਵਿਆ ਹਨ, ਜੋ `TTS_OUTPUT` ਤੋਂ ਨਿਯੰਤ੍ਰਿਤ ਹਨ:

```env
TTS_OUTPUT=local    # PC speaker (ਡਿਫਾਲਟ)
TTS_OUTPUT=remote   # ਕੇਵਲ ਕੈਮਰੇ ਦੀ ਸਪੀਕਰ
TTS_OUTPUT=both     # ਕੈਮਰੇ ਦੀ ਸਪੀਕਰ + PC speaker ਸਾਥੀ ਸਾਥੀ
```

#### A) ਕੈਮਰੇ ਦੀ ਸਪੀਕਰ (go2rtc ਰਾਹੀਂ)

`TTS_OUTPUT=remote` (ਜਾਂ `both`) ਸੈਟ ਕਰੋ। [go2rtc](https://github.com/AlexxIT/go2rtc/releases) ਦੀ ਲੋੜ ਹੈ:

1. [ਰੀਲੀਜ਼ ਪੰਨਾ](https://github.com/AlexxIT/go2rtc/releases) ਤੋਂ ਬਾਈਨਰੀ ਡਾਉਨਲੋਡ ਕਰੋ:
   - Linux/macOS: `go2rtc_linux_amd64` / `go2rtc_darwin_amd64`
   - **Windows: `go2rtc_win64.exe`**

2. ਇਸਨੂੰ ਸਥਿਤੀ ਤੇ ਰੱਖੋ ਅਤੇ ਨਾਮ ਬਦਲੋ:
   ```
   # Linux / macOS
   ~/.cache/embodied-claude/go2rtc/go2rtc          # chmod +x ਦੀ ਲੋੜ ਹੈ

   # Windows
   %USERPROFILE%\.cache\embodied-claude\go2rtc\go2rtc.exe
   ```

3. ਉਸੇ ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ `go2rtc.yaml` ਬਣਾਓ:
   ```yaml
   streams:
     tapo_cam:
       - rtsp://YOUR_CAM_USER:YOUR_CAM_PASS@YOUR_CAM_IP/stream1
   ```
   ਸਥਾਨਕ ਕੈਮਰਾ ਖਾਤਾ ਦੇ ਪ੍ਰਮਾਣ ਪੱਤਰ ਦੀ ਵਰਤੋਂ ਕਰੋ (ਆਪਣਾ TP-Link ਕਲਾਉਡ ਖਾਤਾ ਨਹੀਂ)।

4. familiar-ai ਆਟੋਮੇਟਿਕਲ ਕੰਮ ਦੇ ਦੌਰਾਨ go2rtc ਸ਼ੁਰੂ ਕਰਦੀ ਹੈ। ਜੇ ਤੁਹਾਡਾ ਕੈਮਰਾ ਦੋ-ਤਰੀਕਾ ਆਵਾਜ਼ ਨੂੰ ਸਮਰਥਨ ਕਰਦਾ ਹੈ (ਬੈਕਚੈਨਲ), ਆਵਾਜ਼ ਕੈਮਰੇ ਦੀ ਸਪੀਕਰ ਤੋਂ ਖੇਡਦੀ ਹੈ।

#### B) ਸਥਾਨਕ PC ਸਪੀਕਰ

ਡਿਫਾਲਟ (`TTS_OUTPUT=local`). ਦੌਰਾਨ ਖਿਡਾਰੀਆਂ ਨੂੰ ਕ੍ਰਮ ਵਿੱਚ ਕੋਸ਼ਿਸ਼ ਕਰਦੀ ਹੈ: **paplay** → **mpv** → **ffplay**. ਜਦੋਂ `TTS_OUTPUT=remote` ਅਤੇ go2rtc ਉਪਲਬਧ ਨਹੀਂ ਹੋਵੇ ਤਾਂ ਵੀ ਵਰਤਿਆ ਜਾਂਦਾ ਹੈ।

| OS | ਇੰਸਟਾਲ |
|----|---------|
| macOS | `brew install mpv` |
| Ubuntu / Debian | `sudo apt install mpv` (ਜਾਂ `paplay` ਰਾਹੀਂ `pulseaudio-utils`) |
| WSL2 / WSLg | `sudo apt install pulseaudio-utils` — `.env` ਵਿੱਚ `PULSE_SERVER=unix:/mnt/wslg/PulseServer` ਸੈੱਟ ਕਰੋ |
| Windows | [mpv.io/installation](https://mpv.io/installation/) — ਡਾਉਨਲੋਡ ਕਰੋ ਅਤੇ PATH ਵਿੱਚ ਸ਼ਾਮਿਲ ਕਰੋ, **ਜਾਂ** `winget install ffmpeg` |

> ਜੇਕਰ ਕੋਈ ਆਵਾਜ਼ ਸਜਿਆ ਸਾਜ਼ੀਕਰ ਨਹੀਂ ਹੈ, ਫਿਰ ਵੀ ਬੋਲਣ ਜਾਂਦਾ ਹੈ — ਇਹ ਖੇਡ ਨਹੀਂ ਕਰੇਗਾ।

### ਆਵਾਜ਼ ਇਨਪੁਟ (Realtime STT)

ਸਦਾ-ਚਾਲੂ, ਬਿਨਾਂ-ਹੱਥਾਂ ਦੇ ਆਵਾਜ਼ ਇਨਪੁਟ ਲਈ `.env` ਵਿੱਚ `REALTIME_STT=true` ਸੈਟ ਕਰੋ:

```env
REALTIME_STT=true
ELEVENLABS_API_KEY=sk_...   # TTS ਦੇ ਨਾਲ ਵਧਦਾ ਕੰਜੀ
```

familiar-ai ਮਾਈਕ੍ਰੋਫੋਨ ਦੀ ਆਵਾਜ਼ ਨੂੰ ElevenLabs Scribe v2 'ਤੇ ਸ੍ਰਾਵਿਤ ਕਰਦੀ ਹੈ ਅਤੇ ਤੁਸੀਂ ਬੋਲਣਾ ਰੋਕ ਦਿਓ ਤਾਂ ਐਨਕੋਡ ਦੀ ਆਵਾਜ਼ ਲਗਾਤਾਰ ਕੀਤਾ ਜਾਂਦਾ ਹੈ। ਕੋਈ ਬਟਨ ਦਬਾਉਣ ਦੀ ਲੋੜ ਨਹੀਂ। ਇਹ ਪੁਸ਼-ਟੂ-ਟਾਕ ਮੋਡ (Ctrl+T) ਨਾਲ ਸਾਥੀ ਬਣਦਾ ਹੈ।

## ਪ੍ਰਸ਼ਨ ਪੁੱਛੋ

**Q: ਕੀ ਇਹ GPU ਦੇ ਬਿਨਾਂ ਕੰਮ ਕਰਦਾ ਹੈ?**
ਹਾਂ। ਜੋ embedding model (multilingual-e5-small) CPU 'ਤੇ ਚੰਗਾ ਕੰਮ ਕਰਦਾ ਹੈ। GPU ਇਸਨੂੰ ਤੇਜ ਕਰਦੇ ਹਨ ਪਰ ਲੋੜੀਂਦਾ ਨਹੀਂ ਹੈ।

**Q: ਕੀ ਮੈਂ Tapo ਤੋਂ ਵੱਖਰੇ ਕੈਮਰੇ ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦਾ ਹਾਂ?**
ਕੋਈ ਵੀ ਕੈਮਰਾ ਜੋ ONVIF + RTSP ਦਾ ਸਮਰਥਨ ਕਰਦਾ ਹੈ, ਦਾ ਕੰਮ ਕਰਨਾ ਚਾਹੀਦਾ ਹੈ। Tapo C220 ਹੈ ਜੋ ਅਸੀਂ ਟੈਸਟ ਕੀਤਾ ਹੈ।

**Q: ਕੀ ਮੇਰੇ ਡਾਟਾ ਕਿਸੇ ਜਗ੍ਹਾਂ ਭੇਜੇ ਜਾਂਦੇ ਹਨ?**
ਚਿੱਤਰ ਅਤੇ ਪਾਠ ਤੁਹਾਡੇ ਚੁਣੇ ਹੋਏ LLM API ਨੂੰ ਪ੍ਰੋਸੈਸਿੰਗ ਲਈ ਭੇਜੇ ਜਾਂਦੇ ਹਨ। ਯਾਦਾਂ ਸਥਾਨਕ ਵਿਚ `~/.familiar_ai/` ਵਿੱਚ ਸਟੋਰ ਕੀਤੀਆਂ ਜਾਂਦੀਆਂ ਹਨ।

**Q: ਏਜੈਂਟ `（...）` ਲਿਖਦਾ ਹੈ ਬੋਲੇ ਬਿਨਾਂ?**
ਯਕੀਨੀ ਬਣਾਓ ਕਿ `ELEVENLABS_API_KEY` ਸੈੱਟ ਹੈ। ਇਸਦੇ ਬਿਨਾਂ, ਆਵਾਜ਼ ਅਯੋਗ ਹੈ ਅਤੇ ਏਜੈਂਟ ਟੈਕਸਟ 'ਤੇ ਵਾਪਸੀ ਕਰਦਾ ਹੈ।
