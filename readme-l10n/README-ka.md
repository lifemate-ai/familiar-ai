## MCP рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃў

familiar-ai рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа [MCP (Model Context Protocol)](https://modelcontextprotocol.io) рЃАрЃћрЃарЃЋрЃћрЃарЃА. рЃћрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃгрЃЋрЃЊрЃЮрЃА рЃАрЃљрЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃЮ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃА, рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА, рЃЋрЃћрЃЉрЃА рЃФрЃўрЃћрЃЉрЃўрЃА, рЃљрЃю рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃАрЃ«рЃЋрЃљ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.

рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА `~/.familiar-ai.json` (рЃўрЃЏрЃљрЃЋрЃћ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ, рЃарЃЮрЃњрЃЮрЃарЃф Claude Code):

```json
{
  "mcpServers": {
    "filesystem": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/home/user"]
    },
    "memory": {
      "type": "sse",
      "url": "http://localhost:3000/sse"
    }
  }
}
```

рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЮрЃарЃў рЃАрЃљрЃбрЃарЃљрЃюрЃАрЃърЃЮрЃарЃбрЃЮ рЃбрЃўрЃърЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃџрЃўрЃљ:
- **`stdio`**: рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋ рЃАрЃБрЃЉрЃърЃарЃЮрЃфрЃћрЃАрЃќрЃћ (`command` + `args`)
- **`sse`**: HTTP+SSE рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ (`url`)

рЃњрЃљрЃљрЃДрЃЮрЃцрЃџрЃћрЃЌ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљ `MCP_CONFIG=/path/to/config.json`-рЃќрЃћ.

| PC / Raspberry Pi | рЃњрЃЮрЃюрЃћрЃЉрЃљ | рЃДрЃЋрЃћрЃџрЃљрЃюрЃљрЃўрЃарЃў, рЃарЃљрЃф Python-рЃўрЃА рЃњрЃљрЃгрЃЋрЃЊрЃљрЃА рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃА | **рЃЊрЃўрЃљрЃ«** |

> **рЃЎрЃљрЃЏрЃћрЃарЃљ рЃФрЃџрЃўрЃћрЃа рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ.** рЃЏрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ, familiar-ai рЃЎрЃўрЃЊрЃћрЃЋ рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃАрЃљрЃБрЃЉрЃљрЃарЃў Рђћ рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЋрЃћрЃа рЃ«рЃћрЃЊрЃљрЃЋрЃА рЃЏрЃАрЃЮрЃцрЃџрЃўрЃЮрЃА, рЃарЃљрЃф рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃљрЃЊ рЃЏрЃЌрЃћрЃџрЃў рЃљрЃарЃАрЃћрЃЉрЃљрЃљ.

### рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ (рЃљрЃарЃљрЃарЃўрЃА рЃ«рЃћрЃарЃ«рЃў)

рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃњрЃАрЃБрЃарЃЌ рЃАрЃфрЃљрЃЊрЃЮрЃЌ? рЃЌрЃЦрЃЋрЃћрЃю рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ:

```env
PLATFORM=kimi
API_KEY=sk-...
```

рЃЌрЃљрЃюрЃЊрЃљрЃЌрЃљрЃю рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ `./run.sh` (macOS/Linux/WSL2) рЃљрЃю `run.bat` (Windows) рЃЊрЃљ рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ рЃАрЃљрЃБрЃЉрЃљрЃарЃў. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃ«рЃћрЃарЃ«рЃў рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа.

### Wi-Fi PTZ рЃЎрЃљрЃЏрЃћрЃарЃљ (Tapo C220)

1. Tapo рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃерЃў: **рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў Рєњ рЃЏрЃЮрЃгрЃўрЃюрЃљрЃЋрЃћ Рєњ рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў** Рђћ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў (рЃљрЃа TP-Link рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў)
2. рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃћрЃЌ рЃЎрЃљрЃЏрЃћрЃарЃўрЃА IP рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃќрЃљрЃбрЃЮрЃарЃўрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃерЃў
3. рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `.env`-рЃерЃў:
   ```env
   CAMERA_HOST=192.168.1.xxx
   CAMERA_USER=рЃЌрЃЦрЃЋрЃћрЃюрЃў-рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў-рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў
   CAMERA_PASS=рЃЌрЃЦрЃЋрЃћрЃюрЃў-рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў-рЃърЃљрЃарЃЮрЃџрЃў
   ```

### рЃ«рЃЏрЃљ (ElevenLabs)

1. рЃЏрЃўрЃўрЃдрЃћрЃЌ API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў [elevenlabs.io](https://elevenlabs.io/)
2. рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `.env`-рЃерЃў:
   ```env
   ELEVENLABS_API_KEY=sk_...
   ELEVENLABS_VOICE_ID=...   # рЃљрЃарЃЕрЃћрЃЋрЃўрЃЌрЃў, рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋ рЃ«рЃЏрЃљрЃА рЃЌрЃБ рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃЊрЃљ
   ```

рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЮрЃарЃў рЃњрЃљрЃАрЃЏрЃўрЃА рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃЊрЃћрЃЉрЃљ `TTS_OUTPUT`-рЃўрЃЌ:

```env
TTS_OUTPUT=local    # PC рЃЊрЃўрЃюрЃљрЃЏрЃўрЃЎрЃў (рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў)
TTS_OUTPUT=remote   # рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃЎрЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ
TTS_OUTPUT=both     # рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃЎрЃў + PC рЃЊрЃўрЃюрЃљрЃЏрЃўрЃЎрЃў рЃћрЃарЃЌ рЃЊрЃарЃЮрЃА
```

#### A) рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃЎрЃў (go2rtc-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ)

ЯИЋЯИ▒Я╣ЅЯИЄЯИёЯ╣ѕЯИ▓`TTS_OUTPUT=remote` (рЃљрЃю `both`). рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃА [go2rtc](https://github.com/AlexxIT/go2rtc/releases):

1. рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃЉрЃўрЃюрЃљрЃарЃў [рЃњрЃљрЃЏрЃЪрЃдрЃљрЃЋрЃюрЃћрЃЉрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃЊрЃљрЃю](https://github.com/AlexxIT/go2rtc/releases):
   - Linux/macOS: `go2rtc_linux_amd64` / `go2rtc_darwin_amd64`
   - **Windows: `go2rtc_win64.exe`**

2. рЃЏрЃЮрЃљрЃЌрЃљрЃЋрЃАрЃћрЃЌ рЃЊрЃљ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃЊрЃљрЃљрЃарЃЦрЃЋрЃўрЃЌ:
   ```
   # Linux / macOS
   ~/.cache/embodied-claude/go2rtc/go2rtc          # chmod +x рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ

   # Windows
   %USERPROFILE%\.cache\embodied-claude\go2rtc\go2rtc.exe
   ```

3. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `go2rtc.yaml` рЃўрЃЏрЃљрЃЋрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ:
   ```yaml
   streams:
     tapo_cam:
       - rtsp://YOUR_CAM_USER:YOUR_CAM_PASS@YOUR_CAM_IP/stream1
   ```
   рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў (рЃљрЃа рЃЌрЃЦрЃЋрЃћрЃюрЃў TP-Link рЃдрЃарЃБрЃЉрЃџрЃўрЃА рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў).

4. familiar-ai рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃгрЃДрЃћрЃЉрЃА go2rtc-рЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА. рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЎрЃљрЃЏрЃћрЃарЃљ рЃЮрЃарЃў рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃўрЃА рЃљрЃБрЃЊрЃўрЃЮрЃА (рЃБрЃЎрЃљрЃюрЃљ рЃљрЃарЃ«рЃў) рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ, рЃ«рЃЏрЃљ рЃЏрЃЮрЃЊрЃўрЃА рЃЎрЃљрЃЏрЃћрЃарЃўрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃЎрЃўрЃЊрЃљрЃю.

#### B) рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў PC рЃЊрЃўрЃюрЃљрЃЏрЃўрЃЎрЃў

рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў (`TTS_OUTPUT=local`). рЃфрЃЊрЃўрЃџрЃЮрЃЉрЃА рЃЏрЃфрЃўрЃЋ рЃЏрЃБрЃГрЃљрЃЪрЃћрЃЉрЃџрЃћрЃЉрЃўрЃф: **paplay** Рєњ **mpv** Рєњ **ffplay**. рЃљрЃАрЃћрЃЋрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљFallback, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф `TTS_OUTPUT=remote` рЃЊрЃљ go2rtc рЃљрЃа рЃљрЃарЃўрЃА рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮ.

| OS | рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ |
|----|---------|
| macOS | `brew install mpv` |
| Ubuntu / Debian | `sudo apt install mpv` (рЃљрЃю `paplay` via `pulseaudio-utils`) |
| WSL2 / WSLg | `sudo apt install pulseaudio-utils` Рђћ рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `PULSE_SERVER=unix:/mnt/wslg/PulseServer`-рЃќрЃћ `.env`-рЃерЃў |
| Windows | [mpv.io/рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ](https://mpv.io/installation/) Рђћ рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃЊрЃљ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ PATH-рЃерЃў, **рЃљрЃю** `winget install ffmpeg` |

> рЃЌрЃБ рЃљрЃБрЃЊрЃўрЃЮрЃА рЃЏрЃћрЃЏрЃљрЃарЃЊрЃўрЃЮрЃарЃћрЃЉрЃќрЃћ рЃљрЃарЃљрЃцрЃћрЃарЃў рЃўрЃЏрЃДрЃЮрЃцрЃћрЃЉрЃљ, рЃАрЃљрЃБрЃЉрЃљрЃарЃў рЃЎрЃЋрЃџрЃљрЃЋ рЃњрЃљрЃЏрЃЮрЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ Рђћ рЃўрЃА рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃљрЃа рЃўрЃЌрЃљрЃЏрЃљрЃерЃћрЃЉрЃА.

### рЃ«рЃЏрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃгрЃЋрЃЊрЃЋрЃљ (Realtime STT)

рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `REALTIME_STT=true` `.env`-рЃерЃў рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ рЃ«рЃћрЃџрЃерЃћрЃБрЃ«рЃћрЃЉрЃћрЃџрЃў рЃ«рЃЏрЃўрЃА рЃерЃћрЃгрЃЋрЃЊрЃЋрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА:

```env
REALTIME_STT=true
ELEVENLABS_API_KEY=sk_...   # рЃўрЃњрЃўрЃЋрЃћ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃљ, рЃарЃљрЃф TTS-рЃќрЃћ
```

familiar-ai рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџрЃљрЃЊ рЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃА рЃЏрЃўрЃЎрЃарЃЮрЃцрЃЮрЃюрЃўрЃА рЃљрЃБрЃЊрЃўрЃЮрЃА ElevenLabs Scribe v2-рЃќрЃћ рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃбрЃарЃљрЃюрЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃА, рЃарЃЮрЃфрЃљ рЃАрЃљрЃБрЃЉрЃарЃўрЃА рЃерЃћрЃгрЃДрЃЋрЃћрЃбрЃљ рЃЏрЃЮрЃўрЃАрЃБрЃарЃЋрЃћрЃЉрЃЌ. рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃљрЃГрЃўрЃарЃЮрЃЌ. coexistрЃћрЃЉрЃА push-to-talk рЃарЃћрЃЪрЃўрЃЏрЃЌрЃљрЃю (Ctrl+T).

## TUI

familiar-ai рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃБрЃа UI-рЃА [Textual](https://textual.textualize.io/) рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў рЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў:

- рЃАрЃЦрЃЮрЃџрЃљрЃбрЃў рЃўрЃАрЃбрЃЮрЃарЃўрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ рЃфрЃЮрЃфрЃ«рЃљрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃАрЃбрЃарЃўрЃЏрЃўрЃЌ
- Tab-рЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ `/quit`, `/clear`-рЃўрЃАрЃЌрЃЋрЃўрЃА
- рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃўрЃарЃћрЃЉрЃљ рЃљрЃњрЃћрЃюрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃўрЃџ рЃцрЃўрЃЦрЃарЃА рЃЏрЃЮрЃБрЃАрЃЏрЃўрЃюрЃЮрЃА
- **рЃАрЃљрЃБрЃЉрЃарЃўрЃА рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў** рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ `~/.cache/familiar-ai/chat.log`

рЃџрЃЮрЃњрЃўрЃА рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃўрЃА рЃЊрЃћрЃЋрЃюрЃљ рЃАрЃ«рЃЋрЃљ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў (рЃЏрЃЮрЃАрЃљрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃўрЃљ рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЕрЃљрЃАрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА):
```bash
tail -f ~/.cache/familiar-ai/chat.log
```

## рЃърЃўрЃарЃЮрЃЋрЃюрЃБрЃџрЃў рЃцрЃЮрЃбрЃЮ (ME.md)

рЃЌрЃЦрЃЋрЃћрЃюрЃў familiar-рЃўрЃА рЃърЃўрЃарЃЮрЃЋрЃюрЃћрЃЉрЃљ рЃфрЃ«рЃЮрЃЋрЃарЃЮрЃЉрЃА `ME.md`-рЃерЃў. рЃћрЃА рЃцрЃљрЃўрЃџрЃў рЃљрЃарЃўрЃА gitignored Рђћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃћрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃА.

рЃўрЃ«рЃўрЃџрЃћрЃЌ [`persona-template/en.md`](./persona-template/en.md) рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃљрЃю [`persona-template/ja.md`](./persona-template/ja.md) рЃўрЃљрЃърЃЮрЃюрЃБрЃарЃў рЃЋрЃћрЃарЃАрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.

## рЃ«рЃерЃўрЃарЃљрЃЊ рЃЊрЃљрЃАрЃЏрЃБрЃџрЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃў

**Q: рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ GPU-рЃА рЃњрЃљрЃарЃћрЃерЃћ?**
рЃЊрЃўрЃљрЃ«. рЃћрЃЏрЃЉрЃћрЃЊрЃўрЃюрЃњ рЃЏрЃЮрЃЊрЃћрЃџрЃў (multilingual-e5-small) рЃЏрЃерЃЋрЃћрЃюрЃўрЃЋрЃарЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА CPU-рЃќрЃћ. GPU рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ.

**Q: рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮ рЃАрЃ«рЃЋрЃљ рЃЎрЃљрЃЏрЃћрЃарЃљ Tapo-рЃА рЃњрЃљрЃарЃЊрЃљ?**
рЃљрЃарЃљрЃцрЃћрЃарЃў, рЃарЃЮрЃЏ ONVIF + RTSP-рЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃњрЃљрЃЦрЃЋрЃЌ, рЃБрЃюрЃЊрЃљ рЃўрЃЏрЃБрЃерЃљрЃЮрЃЌ. Tapo C220 рЃљрЃарЃўрЃА рЃўрЃА, рЃарЃљрЃАрЃљрЃф рЃЕрЃЋрЃћрЃю рЃњрЃљрЃАрЃфрЃћрЃЌ.

**Q: рЃЕрЃћрЃЏрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃАрЃљрЃЊрЃЏрЃћ рЃњрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ?**
рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў рЃЊрЃљ рЃбрЃћрЃЦрЃАрЃбрЃў рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃљрЃарЃЕрЃћрЃБрЃџ LLM API-рЃерЃў рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ `~/.familiar_ai/`.

**Q: рЃарЃљрЃбрЃЮрЃЏ рЃгрЃћрЃарЃЊрЃљ рЃљрЃњрЃћрЃюрЃбрЃў `№╝ѕ...№╝Ѕ` рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃарЃЮрЃЏ рЃћрЃА рЃўрЃАрЃљрЃБрЃЉрЃарЃЮрЃА?**
рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ `ELEVENLABS_API_KEY` рЃњрЃљрЃгрЃЋрЃЊрЃўрЃџрЃў рЃљрЃарЃўрЃА. рЃЏрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ, рЃ«рЃЏрЃљ рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃўрЃљ рЃЊрЃљ рЃљрЃњрЃћрЃюрЃбрЃў рЃбрЃћрЃЦрЃАрЃбрЃќрЃћ рЃњрЃљрЃќрЃарЃЊрЃћрЃЏрЃЊрЃћрЃЉрЃљ.

## рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃљрЃЊ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ

рЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЊрЃћрЃЉрЃўрЃЌ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА? рЃюрЃљрЃ«рЃћрЃЌ [docs/technical.md](./docs/technical.md) familiar-ai-рЃўрЃА рЃЎрЃЋрЃџрЃћрЃЋрЃўрЃА рЃЊрЃљ рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ Рђћ ReAct, SayCan, Reflexion, Voyager, рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ.

## рЃЎрЃЮрЃюрЃбрЃарЃўрЃЉрЃБрЃфрЃўрЃарЃћрЃЉрЃљ

familiar-ai рЃљрЃарЃўрЃА рЃдрЃўрЃљ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў. рЃЌрЃБ рЃћрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃљрЃю resonates Рђћ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃљрЃЊ рЃЌрЃБ рЃцрЃўрЃџрЃЮрЃАрЃЮрЃцрЃўрЃБрЃарЃљрЃЊ Рђћ рЃЎрЃЮрЃюрЃбрЃарЃўрЃЉрЃБрЃфрЃўрЃћрЃЉрЃў рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ.

** рЃЎрЃљрЃарЃњрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў рЃЊрЃљрЃАрЃљрЃгрЃДрЃћрЃЉрЃљрЃЊ:**

| рЃбрЃћрЃарЃўрЃбрЃЮрЃарЃўрЃљ | рЃарЃљ рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ |
|------|---------------|
| рЃљрЃ«рЃљрЃџрЃў рЃбрЃћрЃЦрЃюрЃўрЃЎрЃљ | рЃБрЃцрЃарЃЮ рЃЏрЃћрЃбрЃў рЃЎрЃљрЃЏрЃћрЃарЃўрЃА (RTSP, IP Webcam), рЃЏрЃўрЃЎрЃарЃЮрЃцрЃЮрЃюрЃћрЃЉрЃўрЃА, рЃљрЃЦрЃбрЃБрЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ |
| рЃљрЃ«рЃљрЃџрЃў рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃў | рЃЋрЃћрЃЉрЃА рЃФрЃўрЃћрЃЉрЃљ, рЃАрЃљрЃ«рЃџрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃћрЃЉрЃљ, рЃЎрЃљрЃџрЃћрЃюрЃЊрЃљрЃарЃў, рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃарЃљрЃЏ MCP-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ |
| рЃљрЃ«рЃљрЃџрЃў рЃБрЃЎрЃљрЃюрЃљ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃћрЃЉрЃў | рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў LLM рЃљрЃю рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃўрЃљрЃгрЃЋрЃЊрЃЮрЃА `stream_turn` рЃЕрЃљрЃарЃЕрЃЮ |
| рЃърЃўрЃарЃЮрЃЋрЃюрЃБрЃџрЃў рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃў | ME.md рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃюрЃћрЃЉрЃерЃў рЃЊрЃљ рЃърЃўрЃарЃЮрЃЋрЃюрЃћрЃЉрЃћрЃЉрЃќрЃћ |
| рЃЎрЃЋрЃџрЃћрЃЋрЃљ | рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃАрЃБрЃарЃЋрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў, рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃдрЃћрЃЉрЃљ, рЃњрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЌрЃћрЃЮрЃарЃўрЃўрЃА рЃњрЃљрЃгрЃЋрЃЊрЃћрЃЌрЃћрЃЉрЃљ |
| рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ | рЃњрЃљрЃгрЃЋрЃЊрЃћрЃЉрЃў, рЃњрЃљрЃўрЃАрЃљрЃЋрЃАрЃћрЃЉрЃљ, рЃЌрЃљрЃарЃњрЃЏрЃюрЃљ |

рЃўрЃ«рЃўрЃџрЃћрЃЌ [CONTRIBUTING.md](./CONTRIBUTING.md) рЃЊрЃћрЃЋ рЃЏрЃЮрЃърЃЮрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃбрЃўрЃџрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃЊрЃљ PR рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.

рЃЌрЃБ рЃљрЃа рЃ«рЃљрЃарЃЌ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃЊ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃЌ, [рЃдрЃўрЃљ рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃў](https://github.com/lifemate-ai/familiar-ai/issues) Рђћ рЃЉрЃћрЃЊрЃюрЃўрЃћрЃарЃўрЃљ, рЃарЃЮрЃЏ рЃљрЃЏ рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃўрЃЌ рЃАрЃљрЃЏрЃљрЃарЃЌрЃџрЃћрЃЉрЃБрЃџрЃЮрЃЉрЃЌ.
